# Contributing to Yana

First off, thank you for considering contributing to Yana! 

## How to Contribute
1. **Fork** the repository to your own GitHub account.
2. **Clone** the project to your machine.
3. **Create a branch** locally with a succinct but descriptive name (`feat/new-service-page` or `fix/header-alignment`).
4. **Commit** changes to the branch following Conventional Commits guidelines.
5. **Push** changes to your fork.
6. **Open a Pull Request** in our repository and follow the PR template.

## Submitting Issues
* Use the provided Issue Templates for bug reports and feature requests.
* Check existing issues before opening a new one to avoid duplicates.
